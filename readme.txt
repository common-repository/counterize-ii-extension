== Description ==
Counterize II を拡張するプラグイン。アクセスカウンターの表示機能をサイドバーウィジェットとして提供します。前提として Counterize II がインストール・有効化されている必要があります。

== Installation ==
「Counterize II」の公式サイトより「Counterize II」をダウンロードし、事前に「Counterize II」をインストール・適用して下さい。
「Counterize II EXTENSION」をダウンロードし、WordPressにインストール・適用して下さい。
管理画面の [外観] - [ウィジェット] を開き、アクセスカウンターをサイドバーなどにドラッグ＆ドロップして下さい。
